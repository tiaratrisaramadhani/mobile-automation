# 📱 Mobile Automation Testing

This project is a Mobile Automation Testing framework using Appium and WebdriverIO.

---

## 🛠 Tech Stack
- Appium
- WebdriverIO
- TypeScript
- Node.js

---

## 📂 Project Structure
- /test
- /pages
- wdio.conf.ts

---

## 🧪 Test Coverage
- Login testing
- Form validation
- Navigation testing
- UI interaction testing

---

## 🚀 How to Run

npm install  
npm run test

---

## 📊 Reporting
Allure report is used for test result visualization.

Run:
npx allure serve allure-results
